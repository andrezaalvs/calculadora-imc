# Diretório de Imagens

Este diretório contém as imagens (prints de tela) do aplicativo para documentação.

## Como adicionar imagens

1. Cole suas imagens neste diretório
2. Use o formato Markdown no README.md principal:

```markdown
![Descrição da imagem](docs/images/nome-da-imagem.png)
```

## Formatos suportados

- PNG (recomendado para screenshots)
- JPG/JPEG
- GIF
- SVG

## Exemplo de uso no README

```markdown
## Screenshots

### Tela Principal
![Tela Principal do Aplicativo](docs/images/tela-principal.png)

### Tela de Histórico
![Tela de Histórico](docs/images/tela-historico.png)
```
